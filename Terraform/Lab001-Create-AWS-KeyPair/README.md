# Terraform Lab001 - Create AWS Key Pair

## Objective
Create an AWS EC2 Key Pair using Terraform.

## Technologies Used
- Terraform
- AWS
- TLS Provider
- Local Provider

## Files
- `main.tf` - Terraform configuration for creating the AWS key pair.

## Commands Used

```bash
terraform init
terraform plan
terraform apply
terraform destroy
```

## What I Learned
- How to configure Terraform providers.
- How to define resources in Terraform.
- How to generate an SSH key pair using the TLS provider.
- How to create an AWS EC2 Key Pair.
- How to save the private key locally.

## Outcome
Successfully created an AWS EC2 Key Pair using Terraform and understood the Terraform workflow.
